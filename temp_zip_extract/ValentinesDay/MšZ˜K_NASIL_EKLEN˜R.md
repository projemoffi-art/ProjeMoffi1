# Müzik Nasıl Eklenir? 🎵

## Sezen Aksu - Ey Aşk Şarkısını Eklemek İçin:

### Yöntem 1: MP3 Dosyasını Yükle
1. **Sezen Aksu - Ey Aşk** şarkısını bilgisayarına indir
2. Dosyayı şu klasöre kopyala:
   ```
   ValentinesDay\public\music\
   ```
3. Dosya adını değiştir: `ey-ask.mp3`
4. Sayfayı yenile - müzik otomatik çalacak!

### Yöntem 2: Şu An Çalışan Müzikler
Kod 3 farklı romantik müzik ile geldi (yedek olarak):
- Bensound - Slow Motion
- Bensound - Romantic
- Bensound - Memories

### Müzik Çalmazsa:
1. **Sağ alttaki müzik butonuna tıkla**
2. Tarayıcı izin penceresi açarsa **"İzin Ver"** seç
3. Eğer hala çalmazsa **F12** tuşuna bas, Console'da hata var mı kontrol et

### Test Et:
http://localhost:5174 adresini aç ve müzik butonuna tıkla!

Müzik çalıyorsa buton **altın renkte** parlayacak! ✨
